# Personal Website

This project is a personal website built with React + TypeScript + Vite.
It includes:

- A home page
- A resume section
- A projects section
- Runtime content loaded from `public/data/resume.json`

## Local development

```bash
npm install
npm run start
```

## Production build

```bash
npm run build
npm run preview
```

## Content updates

Edit `public/data/resume.json` to update profile, resume, and projects content
without touching application code.
